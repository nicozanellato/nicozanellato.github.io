Projects

This is a selection of data science and analytics projects showcasing my skills and interests. I mainly code in python as it is an open source and very versatile coding language that can be used in a variety of projects and tasks such as data manipulation, predictive analytics, data science and web development. Tableau is my reporting tool of choice as it has more functionality and aesthetically more pleasing than other software that I used such as Power BI and Google Data Studio. Scraping for Beers

This was my first data science project, scraping data about craft beers, visualise it and then applying machine learning to predict overall score based on specific features.
Technologies: pandas, Beautifulsoup, scikit-learn with K-means clustering and Tableau
