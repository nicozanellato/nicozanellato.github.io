Scraping Beers

This is my first data science project, scraping craft beer data and applying machine learning.

Being a craft beer lover I started using the app Untappd that allows you to rate and share the beers you’re drinking with your mates and see interesting breweries, new beers and events. As it is an app used all over the world, breweries and pubs list their beers on it and people can rate them. I also found the app very useful for when I am travelling. It highlights local breweries and pubs and their beer listing on the map, so that you can find and drink some local goodness.

I thought it would be amazing if I could get some of their data, create some data visualisation and apply machine learning on it! Also, I wanted to work on something that I love and stem away from the box standard datasets used in data science tutorials such as Iris. There is nothing wrong with the Iris dataset, it’s a great way to start per se, but I wanted to do something a bit more original and creative.

Untppd also has a website, so I decided to give web scraping a go, by scraping all the top 50 breweries and their beer listings for every country. I managed to scrape about 40k different beers from all over the world! After scraping the website, I have created an interactive visualisation in Tableau and uploaded it on my Tableau Public profile, which can be seen here below. More info on how I went about scraping the website and applied machine learning can be found after the dashboard. Full code available on my GitHub page.


I have applied K-means clustering algorithm to the the dataset to predict the beer rating of a new beer, given the brewery, ABV (alcohol content), IBU (International Bitterness Unit) and beer type. I managed to reach an accuracy score of 88%! Not too bad really! Feedbacks are welcome! Please let me know what you think or what can be improved.
